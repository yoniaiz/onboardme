## Visual Design Notes

This game is about building a connected trail through the code. Visual elements should emphasize:

- **Breadcrumb/trail metaphor** - path building, connections
- **Progressive reveal** - steps fill in as completed
- **Connection lines** - show how layers connect
- **Detective/investigation feel** - following clues through code
