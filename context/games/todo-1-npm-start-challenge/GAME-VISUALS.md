# TODO #1: npm start --challenge - Visuals

## Visual Design Notes

This game is about bringing the project to life and interacting with it. Visual elements should emphasize:

- **Launch/startup feeling** - rocket, ignition, power-on aesthetics
- **Achievement badges** for each completed challenge
- **Progress through challenges** shown as a checklist
- **Split focus** - CLI gives tasks, player works in browser/terminal
