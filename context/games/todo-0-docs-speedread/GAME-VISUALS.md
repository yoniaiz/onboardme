# TODO #0: docs --speedread - Visuals

## Visual Design Notes

This game focuses on timed information hunting. Visual elements should emphasize:

- **Clean target display** - The question should be prominent and clear
- **Visible timer** - Always showing, color changes as time runs low
- **Progress indicator** - Show 1/5, 2/5, etc. progression
- **Celebration on success** - Points floating up, speed bonus display
- **Learning on failure** - Show answer location clearly
