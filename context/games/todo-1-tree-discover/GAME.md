# TODO #1: tree --discover

## Game Overview

**Type:** Progressive reveal  
**Goal:** Uncover project structure through exploration  
**Level:** 1 (TODO #1)  
**Sub-task:** First sub-task of TODO #1

## Description

This game teaches players to build a mental map of what exists in the codebase. Through progressive exploration, players uncover the project structure, learning about:

- Project language and framework
- Directory structure
- What services exist
- How to run the project

## Learning Outcomes

- Understanding project language and framework
- Familiarity with directory structure
- Knowledge of what services exist
- Understanding how to run the project

## Game Flow

Players progressively reveal the project structure through exploration. Each discovery unlocks more information about the codebase, building a comprehensive understanding of what exists before moving on to navigation and deeper understanding.
