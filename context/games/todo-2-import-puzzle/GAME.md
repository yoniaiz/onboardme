## Game Overview

**Type:** Ordering  
**Goal:** Understand import patterns and conventions  
**Level:** 2 (TODO #2)  
**Sub-task:** Second sub-task of TODO #2

## Description

This game teaches players to understand import patterns and conventions in the codebase. Players learn:

- How imports are structured
- Import ordering conventions
- Dependency relationships through imports
- Module organization patterns

## Learning Outcomes

- How to search effectively
- Understanding naming conventions
- Knowledge of import structure
- Familiarity with file organization patterns

## Game Flow

Players are presented with import statements or module relationships and must order them correctly or identify the proper import pattern. This builds understanding of how the codebase organizes its modules and dependencies.
