# TODO #2: grep --hunt

## Game Overview

**Type:** Timed search  
**Goal:** Find specific code from symptom descriptions  
**Level:** 2 (TODO #2)  
**Sub-task:** First sub-task of TODO #2

## Description

This game teaches players to navigate the codebase effectively by finding specific code based on symptom descriptions. Players learn:

- How to search effectively
- Naming conventions
- Import structure
- File organization patterns

## Learning Outcomes

- How to search effectively
- Understanding naming conventions
- Knowledge of import structure
- Familiarity with file organization patterns

## Game Flow

Players are given symptom descriptions (like bug reports or user complaints) and must find the relevant code that handles those scenarios. This teaches effective codebase navigation and search strategies.
