## Visual Design Notes

This game is about hunting bugs through failing tests. Visual elements should emphasize:

- **Red/failure aesthetic** for the failing test
- **Detective/hunt feeling** - tracking down the bug
- **Timer tension** - urgency without panic
- **Victory moment** when bug is found
