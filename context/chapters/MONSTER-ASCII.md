## Design Philosophy

When rendering the Monster in responses:

1. **Match health to mood** â€” Use the appropriate ASCII state
2. **Sound effects precede dialogue** â€” Always start with *kzzzt* or similar
3. **End with signal tag** â€” Close transmissions with bracketed status
4. **Spaghetti = chaos** â€” More spaghetti tentacles = more chaotic/powerful
5. **Clean = peaceful** â€” Documented Monster has no spaghetti, clean lines
